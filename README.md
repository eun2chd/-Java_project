# 공피하기 게임 프로젝트 소개 #
# 개발환경 #
  - windows10
  - JAVA
---
# 게임 설명 #
 - 플레이어를 방향키로 이동이 가능하다.
 - 플레이어의 체력이 0이 되면 게임 Thread 가 멈추고 다시시작 문구가 표시된다.
 - SCORE 의 점수가 일정점수에 도달하게 되면 게임의 난이도가 올라간다.
    - 공의 스피드 증가
    - 공의 갯수 증가
 - 최종 목표 : 최대한 오랫동안 공을 피해서 높은 SCORE 을 얻는다.
